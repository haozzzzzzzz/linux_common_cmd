# 服务事故及服务不可用定级



##事故分级

极严重 > 严重 > 一般 > 轻微

|        | 涉及范围                 | 时间跨度   | 是否涉及宕机 | 是否影响用户体验 | 是否影响产品功能 | 是否可在1小时内恢复 |
| ------ | ------------------------ | ---------- | ------------ | ---------------- | ---------------- | ------------------- |
| 极严重 | 大部分用户 (60%请求以上) | 2小时以上  | 是           | 是               | 是               | 否                  |
| 严重   | 部分用户 (30%~60%请求)   | 1~2小时    | 是           | 是               | 是               | 否                  |
| 一般   | 小部分用户 (10%~30%请求) | 1小时以下  | 否           | 是               | 是               | 是                  |
| 轻微   | 个别用户 (10%以下请求)   | 30分钟以内 | 否           | 是               | 否               | 是                  |
